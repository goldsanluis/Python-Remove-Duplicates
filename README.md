# Remove Duplicates

A Python function that filters out duplicate elements from a list
while keeping the first occurrence of each item in its original order.

## What It Does
- Preserves the order of first appearances
- Works with both numbers and strings
- Raises a TypeError if input is not a list

## Sample Output
remove_duplicates([1, 2, 2, 3, 3, 3, 4])  # [1, 2, 3, 4]
remove_duplicates(['a', 'b', 'a', 'c'])    # ['a', 'b', 'c']
remove_duplicates([1, 1, 1])               # [1]

## Requirements
- Python 3.x
